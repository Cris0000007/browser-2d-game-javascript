# Browser-Based 2D Game Project

A browser-based 2D game developed using JavaScript, p5.js, and Matter.js. This project demonstrates basic game logic, player interaction, physics-based movement, collision handling, and visual game design.

## Project Overview

This project is a simple 2D browser game built using JavaScript. The game runs directly in the web browser and uses p5.js for drawing and interaction, together with Matter.js for physics-related behaviour.

The project focuses on applying programming concepts such as event handling, collision detection, game object movement, visual design, and interactive gameplay.

## File Description

- `index.html`  
  Loads the game files and required JavaScript libraries.

- `sketch.js`  
  Contains the main game logic, drawing functions, user interaction, object movement, and collision handling.

- `libraries/`  
  Contains the p5.js, p5.sound, and Matter.js libraries required to run the project.

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double click `index.html`.
4. The game will run directly in the web browser.

Alternatively, the project can be opened using a local development server.
